### codeit 온라인 강의 수강

* Next.js 기초 학습

* Next.js는 풀스택 웹 어플리케이션을 구축하기 위한 React 프레임워크

  프론트 단의 UI구성은 React로 하고, Next.js를 사용하여 추가 기능과 최적화를 수행

  SSR(서버 사이드 렌더링)으로 모든 페이지에 대한 HTML 마크업이 서버에서 수행, 페이지 로딩 속도를 크게 개선

### 빌드
* npm install

### 개발 모드 실행
* npm run dev

### 일반 모드 실행
* npm run build
* npm run start
